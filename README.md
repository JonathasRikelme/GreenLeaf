# GreenLeaf Delivery 🥗 - Projeto Acadêmico

Este é um aplicativo de delivery de comida saudável desenvolvido como um protótipo acadêmico (nível Júnior).

## 🚀 Tecnologias Utilizadas
- **React Native** (Framework principal)
- **Expo** (Facilitador de desenvolvimento)
- **React Navigation** (Navegação Stack e Tabs)
- **Expo Location** (Sensor de GPS)
- **Expo Notifications** (Sensor de Notificações Push Locais)

## 📱 Funcionalidades
1. **Login Simples**: Acesso simulado ao aplicativo.
2. **Cardápio (FlatList)**: Exibição de 10 pratos saudáveis com fotos e preços.
3. **Detalhes do Prato**: Informações completas sobre cada item.
4. **GPS (Sensor)**: Obtenção de localização atual via GPS do dispositivo.
5. **Notificações (Sensor)**: Envio de alerta local simulando status de pedido.

## 📖 Estrutura do Código
O código foi escrito de forma simples e **totalmente comentado linha por linha** para fins didáticos, explicando cada função e componente utilizado.

## 🛠️ Como rodar o projeto
1. Tenha o **Node.js** instalado.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o Expo:
   ```bash
   npx expo start
   ```
4. Use o aplicativo **Expo Go** no seu celular para escanear o QR Code e visualizar o app.
